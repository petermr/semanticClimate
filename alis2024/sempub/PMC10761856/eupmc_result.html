'2045-2322'}} 2024 256 There is a fervent debate about the processes underpinning false memories formation. Seminal theories have suggested that semantic memory would be involved in false memories production, while episodic memory would counter their formation. Yet, direct evidence corroborating such view is still lacking. Here, we tested this possibility by asking participants to perform the Deese-Roediger-McDermott (DRM) task, a typical false memory paradigm, in which they had to study lists of words and subsequently to recognize and distinguish them from new words (i.e., the false memory items). The same participants were also required to perform a semantic task and an episodic-source memory task. Our results showed that a higher number of false memories in the DRM task occurred for those participants with better semantic memory abilities, while a lower number of false memories occurred for participants with better episodic abilities. These findings support a key role of semantic processes in false memory formation and, more generally, help clarify the specific contribution of different memory systems to false recognitions.
